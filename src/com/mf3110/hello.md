## git 명령어 요약

 - clone : 원격 저장소 복사
 - add   : 스테이지 영역에 작업 파일 추가
 - commit: 세이브, 스테이지 영역의 파일들을 가지고 커밋(=세이브)을 할 수 있다.
 - push  : 원격 저장소에 커밋을 업로드 한다.
 - revert : 마지막 커밋 상태로 되돌아간다.
